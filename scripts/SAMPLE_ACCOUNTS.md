## 로그인 테스트 방법

1. **회원가입으로 계정 생성**:
   - http://localhost:3000/auth/signup 접속
   - 위 샘플 계정 정보로 회원가입

2. **로그인 테스트**:
   - http://localhost:3000/auth/login 접속
   - 이메일과 비밀번호 입력
   - 로그인 성공 시 대시보드로 이동

3. **관리자 기능 테스트**:
   - admin@aidaily.com으로 로그인
   - /admin/dashboard로 자동 리디렉션

4. **일반 사용자 기능 테스트**:
   - demo@aidaily.com으로 로그인
   - /dashboard로 이동
   - 뉴스 콘텐츠, 고객 관리, 텔레그램 설정 등 확인
